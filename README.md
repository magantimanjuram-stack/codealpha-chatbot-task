# Simple Chatbot 🤖

This is a basic chatbot built using Python.

## 📌 Features
- Responds to greetings like "hello" and "hi"
- Answers simple questions
- Exits when user types "bye"

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Download the file
2. Open terminal
3. Run:
   python chatbot.py

## 📚 Concepts Used
- if-else conditions
- loops
- user input handling
